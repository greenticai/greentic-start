# Security Policy

Greentic takes security vulnerabilities seriously and appreciates responsible disclosure.

## Supported Versions

| Version Range | Supported |
|--------------|-----------|
| `0.4.x` (current stable) | Yes |
| `0.3.x` | No |
| `< 0.3.x` | No |

## Reporting a Vulnerability

Do not open a public issue for security reports.

Send details to:

`security@greentic.ai`

Please include:

- Vulnerability description
- Affected repository/version
- Reproduction steps or PoC
- Impact assessment

## Disclosure Process

- Acknowledgement target: within 3 business days
- Ongoing status updates during triage/remediation
- Coordinated disclosure after fix for supported versions
